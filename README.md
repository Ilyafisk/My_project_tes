# Telegram Bot - Медитация с проверкой подписки

Telegram-бот на Python для распространения аудиомедитации с обязательной подпиской на закрытый канал.

## Функционал

1. Генерация уникальной одноразовой ссылки-приглашения для каждого пользователя
2. Приветственное сообщение с описанием и кнопкой подписки на канал
3. Проверка подписки пользователя на закрытый канал
4. Отправка аудиофайла с медитацией только подписанным пользователям

## Установка

### 1. Установите Python

Убедитесь, что у вас установлен Python 3.8 или выше:

```bash
python3 --version
```

### 2. Создайте виртуальное окружение

```bash
python3 -m venv venv
```

Активируйте виртуальное окружение:

**На macOS/Linux:**
```bash
source venv/bin/activate
```

**На Windows:**
```bash
venv\Scripts\activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте Telegram бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен бота
4. Сохраните токен - он понадобится для настройки

### 5. Подготовьте закрытый канал

1. Создайте закрытый канал в Telegram
2. **Добавьте бота в администраторы канала (обязательно!)**
3. **Дайте боту права на создание ссылок-приглашений** (в настройках администратора включите "Приглашение пользователей через ссылку")
4. Получите ID канала:
   - Если у канала есть username: используйте `@username`
   - Если нет username: используйте бота [@getmyid_bot](https://t.me/getmyid_bot)

⚠️ **Важно**: Теперь бот автоматически генерирует одноразовые ссылки-приглашения для каждого пользователя, поэтому создавать ссылку вручную не нужно!

### 6. Настройте переменные окружения

1. Скопируйте файл [.env.example](.env.example) в `.env`:

```bash
cp .env.example .env
```

2. Откройте файл `.env` и заполните:

```env
BOT_TOKEN=ваш_токен_бота_от_BotFather
CHANNEL_ID=@your_channel_username_или_числовой_ID
```

⚠️ **Обратите внимание**: Переменная `CHANNEL_INVITE_LINK` больше не нужна - бот генерирует уникальные ссылки автоматически!

### 7. Добавьте аудиофайл

Поместите файл медитации в папку `audio/` с именем `meditation.mp3`:

```bash
# Создайте папку audio (если её нет)
mkdir -p audio

# Скопируйте ваш аудиофайл
cp /путь/к/вашему/файлу.mp3 audio/meditation.mp3
```

## Запуск

Убедитесь, что виртуальное окружение активировано, затем запустите:

```bash
python bot.py
```

Или:

```bash
python3 bot.py
```

Вы увидите сообщение:
```
✅ Бот запущен и готов к работе
```

## Использование бота

1. Пользователь отправляет `/start`
2. Бот генерирует **уникальную одноразовую ссылку-приглашение** для этого пользователя
3. Бот отправляет приветственное сообщение с кнопкой подписки (с уникальной ссылкой)
4. Пользователь переходит по ссылке и подписывается на канал
5. Пользователь нажимает "Проверить подписку"
6. Если подписка подтверждена - появляется кнопка "Получить медитацию"
7. После нажатия на кнопку - бот отправляет аудиофайл

🔐 **Преимущества одноразовых ссылок:**
- Каждая ссылка работает только для одного пользователя
- Невозможно массово поделиться ссылкой
- Полный контроль над тем, кто получает доступ

## Важные моменты

### ⚠️ Бот ДОЛЖЕН быть администратором канала

Обязательно добавьте бота в администраторы канала, иначе он не сможет проверять подписки и создавать ссылки-приглашения.

Как добавить:
1. Зайдите в настройки канала
2. Администраторы → Добавить администратора
3. Найдите вашего бота по username
4. Включите необходимые права (см. ниже)

### Права администратора для бота

Боту ОБЯЗАТЕЛЬНО нужны следующие права:
- ✅ **Приглашение пользователей через ссылку** (ОБЯЗАТЕЛЬНО для генерации ссылок!)
- ✅ Просмотр сообщений
- ❌ Публикация сообщений (не обязательно)
- ❌ Редактирование сообщений (не обязательно)
- ❌ Удаление сообщений (не обязательно)
- ❌ Добавление участников (не обязательно)

### Получение числового ID канала

Если у канала нет username:
1. Перешлите любое сообщение из канала боту [@getmyid_bot](https://t.me/getmyid_bot)
2. Бот покажет ID канала в формате `-1001234567890`
3. Используйте этот ID в `.env` файле

## Устранение неполадок

### Бот не проверяет подписку

- ✅ Убедитесь, что бот добавлен в **администраторы** канала
- ✅ Проверьте правильность `CHANNEL_ID` в [.env](.env)
- ✅ Для приватных каналов без username используйте числовой ID

### Аудиофайл не отправляется

- ✅ Проверьте наличие файла `audio/meditation.mp3`
- ✅ Убедитесь, что файл в формате MP3
- ✅ Проверьте размер файла (Telegram ограничивает 50MB для ботов)
- ✅ Убедитесь, что у бота есть права на чтение файла

### Бот не запускается

- ✅ Проверьте правильность `BOT_TOKEN` в [.env](.env)
- ✅ Убедитесь, что виртуальное окружение активировано
- ✅ Убедитесь, что все зависимости установлены (`pip install -r requirements.txt`)
- ✅ Проверьте версию Python (нужна 3.8+)

### Ошибка "BOT_TOKEN не установлен"

- ✅ Убедитесь, что создан файл `.env` (а не только `.env.example`)
- ✅ Проверьте, что в `.env` нет лишних пробелов вокруг знака `=`
- ✅ Правильный формат: `BOT_TOKEN=123456:ABC-DEF...`

## Логи

Бот выводит подробные логи в консоль:

- `❌` - Пользователь не подписан
- `🎧` - Отправка медитации
